# Security policy

## Supported version

Security fixes are applied to the latest version on the default branch.

## Reporting a vulnerability

Please do not open a public issue for a suspected vulnerability. Use the
repository's **Security → Report a vulnerability** flow to create a private
GitHub security advisory.

Include reproduction steps, affected inputs or URLs, potential impact, and any
suggested mitigation. You should receive an initial response through GitHub
within seven days.

This project does not collect credentials, place trades, or connect to brokerage
accounts. Reports involving third-party exchanges or sportsbooks should also be
sent to the affected provider.
